# NIST COBOL 85 Test Suite – All Programs

Downloaded from: https://gnucobol.sourceforge.io/files/newcob.val.tar.gz
(Original NIST CCVS85 / ANSI85 test suite; NIST direct URL is no longer serving the file.)

Total: 459 COBOL programs (plus EXEC85 driver and CLBRY copybook/library entries).

Programs are stored in the single archive file `newcob.val`. They are organized by
ANSI 1985 COBOL module. Naming: <MODULE><nnn><suffix>, e.g. NC101A, SQ201M.

Module key:
  NC = Nucleus (core language)
  SM = COPY / copy sentence
  IC = CALL sentence
  SQ = Sequential file I-O
  RL = Relative file I-O
  IX = Indexed file I-O
  ST = SORT sentence
  SG = Segments
  OB = Obsolete facilities
  IF = Intrinsic functions
  RW = REPORT section
  DB = Debugging facilities
  CM = Communication section
